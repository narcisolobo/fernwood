# Initial Flows

## Member-facing flow

1. Discover studio → branded website/portal
2. Sign up → create account, sign waiver
3. Browse class schedule → filter by type/instructor/time
4. Book a class → pay (drop-in, class pack, or membership) or join waitlist if full
5. Get reminders → confirmation + pre-class notification
6. Check in → self-service kiosk or app at the studio
7. Manage account → view pack/membership balance, cancel/reschedule, update payment method

## Admin/staff flow

1. Studio setup → classes, instructors, pricing tiers (drop-in/pack/membership), studio policies
2. Daily scheduling → create/edit class calendar, assign instructors, set capacity
3. Member management → view profiles, attendance history, payment status
4. Front-desk ops → manual check-in override, walk-in booking, roster view per class
5. Billing → handle failed payments, refunds, upgrades/downgrades
6. Waitlist handling → auto-fill cancelled spots (usually automated, but admin can see/override)
7. Communications → send announcements, class cancellation alerts
8. Reporting/insights → attendance trends, revenue, at-risk/churning member flags
